2. Texto y Contenedores

Estas etiquetas se usan para estructurar y dar significado semántico al texto y para agrupar contenido relacionado.

Etiqueta Funcionalidad y Ejemplo
<h1> a <h6>Definen encabezados o títulos, de mayor a menor importancia.
<p>Define un párrafo de texto.
<div>Contenedor de bloque. Se usa para agrupar elementos y aplicar estilos. Ocupa todo el ancho.
<span>Contenedor en línea. Se usa para agrupar una parte de texto y aplicar un estilo específico.
<a>Crea un hipervínculo (link). El atributo href especifica la URL.
Ej: <a href="https://google.com">Ir a Google</a>

Ejemplo Práctico de Texto y Contenedores

En el siguiente ejemplo, un <div> agrupa un encabezado, párrafos, un <span> para estilizar texto y un enlace <a>.

Encabezado Principal (h1)

Esto es un Subtítulo (h2)

Este es un párrafo de texto. Los contenedores <div> son de bloque, lo que significa que ocupan todo el ancho disponible, como este recuadro completo que agrupa todo el ejemplo.

Dentro de un párrafo, podemos usar <span> para aplicar un estilo a una parte específica del texto sin afectar al resto. También podemos agregar hipervínculos, como este que te lleva a Google.

Este es el código HTML que genera el ejemplo anterior:

<!-- El <div> agrupa todo el contenido en un solo bloque -->
<div>
    <!-- <h1> es el encabezado de mayor jerarquía -->
    <h1>Encabezado Principal (h1)</h1>
    <h2>Esto es un Subtítulo (h2)</h2>

    <!-- <p> se usa para párrafos de texto -->
    <p>
        Este es un párrafo de texto. Los contenedores <code>&lt;div&gt;</code> son de <strong>bloque</strong>...
    </p>
    <p>
        Dentro de un párrafo, podemos usar <code>&lt;span&gt;</code> para aplicar un estilo a una 
        <!-- El <span> solo afecta a este fragmento de texto -->
        <span style="color: #1abc9c; font-weight: bold;">parte específica del texto</span> 
        sin afectar al resto. También podemos agregar hipervínculos, como este que te lleva a 
        <!-- El <a> crea un enlace a la URL en su atributo href -->
        <a href="https://google.com" target="_blank">Google</a>.
    </p>
</div>
