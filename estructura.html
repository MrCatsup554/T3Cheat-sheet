1. Estructura del Documento y Layout

Estas etiquetas definen la estructura fundamental y las grandes secciones de contenido de una página web.

Etiqueta Funcionalidad
<!DOCTYPE html>Declara que el documento es de tipo HTML5. Es la primera línea, obligatoria.
<html>Etiqueta raíz que engloba todo el contenido de la página.
<head>Contiene "metadatos" sobre la página (título, CSS, etc.). No es visible.
<body>Contiene todo el contenido visible de la página.
<title>Define el título que aparece en la pestaña del navegador.
<header>Contenedor para el encabezado de una página o sección (logo, título, nav).
<nav>Define un bloque de enlaces de navegación principal (menú).
<main>Representa el contenido principal y único del documento.
<aside>Contenido secundario o relacionado, como una barra lateral (sidebar).
<footer>Pie de página de un documento o sección (autor, copyright, contacto).

Ejemplo Práctico de Layout

Las etiquetas <!DOCTYPE html>, <html>, <head> y <body> conforman el esqueleto fundamental de cualquier página. El siguiente código muestra cómo las etiquetas semánticas como <header>, <nav>, <main>, <aside>, y <footer> se organizan dentro del <body> para darle una estructura clara y significativa al contenido. 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
</head>
<body>

    <!-- Encabezado del sitio -->
    <header>
        <h1>Logo y Título Principal del Sitio</h1>
        <nav>
            <a href="/inicio">Inicio</a>
            <a href="/servicios">Servicios</a>
            <a href="/contacto">Contacto</a>
        </nav>
    </header>

    <!-- Contenido principal -->
    <main>
        <h2>Artículo Principal</h2>
        <p>Aquí va el contenido más importante, como el artículo de un blog, una noticia, o la descripción de un producto.</p>
    </main>

    <!-- Contenido secundario o relacionado -->
    <aside>
        <h3>Barra Lateral</h3>
        <p>Contenido secundario: enlaces de interés, publicidad, etc.</p>
    </aside>

    <!-- Pie de página -->
    <footer>
        <p>Copyright © 2025 - Información de contacto.</p>
    </footer>

</body>
</html>
