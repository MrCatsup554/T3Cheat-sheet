4. Tablas

Las tablas se usan para mostrar datos organizados en filas y columnas.

Etiqueta Funcionalidad
<table>Define el contenedor principal de la tabla.
<thead>Agrupa el contenido de la cabecera de la tabla.
<tbody>Agrupa el contenido del cuerpo de la tabla.
<tfoot>Agrupa el contenido del pie de la tabla.
<tr>Table Row. Define una fila en la tabla.
<th>Table Header. Define una celda de encabezado.
<td>Table Data. Define una celda de datos normal.

Ejemplo Práctico de una Tabla

A continuación, una tabla que organiza lenguajes de programación. Fíjate cómo <thead>, <tbody>, y <tfoot> estructuran el contenido.

Lenguaje de Programación Creador Año de Creación
JavaScript Brendan Eich 1995
Python Guido van Rossum 1991
HTML Tim Berners-Lee 1993
Fuente: Datos históricos de la programación.

Este es el código HTML que genera la tabla anterior:

<table>
    <!-- thead agrupa los encabezados de la tabla -->
    <thead>
        <!-- tr define una fila (Table Row) -->
        <tr>
            <!-- th define una celda de encabezado (Table Header) -->
            <th>Lenguaje de Programación</th>
            <th>Creador</th>
            <th>Año de Creación</th>
        </tr>
    </thead>

    <!-- tbody agrupa el cuerpo principal de la tabla -->
    <tbody>
        <tr>
            <!-- td define una celda de datos (Table Data) -->
            <td>JavaScript</td>
            <td>Brendan Eich</td>
            <td>1995</td>
        </tr>
        <tr>
            <td>Python</td>
            <td>Guido van Rossum</td>
            <td>1991</td>
        </tr>
    </tbody>

    <!-- tfoot agrupa el pie de la tabla -->
    <tfoot>
        <tr>
            <!-- colspan="3" hace que la celda ocupe 3 columnas -->
            <td colspan="3">Fuente: Datos históricos.</td>
        </tr>
    </tfoot>
</table>
