6. Formularios

Los formularios son esenciales para la interacción con el usuario, permitiendo recoger datos como en inicios de sesión, encuestas o páginas de contacto.

Etiqueta Funcionalidad
<form>Contenedor para los controles de un formulario.
<label>Define una etiqueta de texto para un control de formulario. Mejora la accesibilidad.
<input>El elemento más versátil. Su comportamiento cambia con el atributo type (ej: text, password, checkbox, radio, submit).
<select>Crea una lista desplegable (dropdown).
<option>Define cada opción seleccionable dentro de un <select>.
<textarea>Crea un campo de texto de múltiples líneas.
