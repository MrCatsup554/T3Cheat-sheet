3. Listas

Las listas se usan para agrupar un conjunto de elementos relacionados.

Etiqueta Funcionalidad
<ul>Unordered List (Lista no ordenada). Usa viñetas.
<ol>Ordered List (Lista ordenada). Usa números o letras.
<li>List Item. Define cada elemento dentro de una lista <ul> u <ol>.
<dl>Description List (Lista de descripciones).
<dt>Description Term. El término o nombre a definir dentro de una <dl>.
<dd>Description Details. La descripción del término (<dt>).

Ejemplos Prácticos de Listas

Lista No Ordenada (<ul>)

Ideal para elementos donde el orden no es importante, como una lista de compras.

  • Café
  • Leche
  • Pan
<ul>
    <li>Café</li>
    <li>Leche</li>
    <li>Pan</li>
</ul>

Lista Ordenada (<ol>)

Se usa cuando la secuencia de los elementos es relevante, como en los pasos de una receta.

  1. Hervir agua
  2. Añadir el café
  3. Servir en una taza
<ol>
    <li>Hervir agua</li>
    <li>Añadir el café</li>
    <li>Servir en una taza</li>
</ol>

Lista de Descripción (<dl>)

Perfecta para glosarios o para definir parejas de término y descripción.

HTML
Lenguaje de Marcado de Hipertexto, usado para estructurar el contenido web.
CSS
Hojas de Estilo en Cascada, usado para diseñar y dar estilo visual al contenido web.
<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de Marcado de Hipertexto, usado para estructurar el contenido web.</dd>
    <dt>CSS</dt>
    <dd>Hojas de Estilo en Cascada, usado para diseñar y dar estilo visual al contenido web.</dd>
</dl>
